This repository holds my personal NixOS configuration, managed with flakes for fully reproducible, declarative system management across two machines: a ThinkPad X220 and an AMD desktop. Host-specific files handle hardware and boot configuration, while a shared base covers services, user preferences, and home-manager integration — ensuring a consistent environment across different hardware.

** Hosts

*** ThinkPad X220
Classic Intel laptop. Legacy GRUB on =/dev/sda=, latest kernel. Runs xmonad as the window manager with xmobar as the status bar.

To apply:
#+BEGIN_SRC bash
sudo nixos-rebuild switch --flake .#thinkpad-x220
home-manager switch --flake .#xmonad
#+END_SRC

*** navi (AMD Desktop)
AMD CPU + NVIDIA GPU desktop. UEFI with systemd-boot, latest kernel. Runs Sway (Wayland). NVIDIA is configured for Wayland via DRM KMS, Vulkan renderer, and the GBM provider. Also runs a Jellyfin media server accessible via Tailscale/LAN.

To apply:
#+BEGIN_SRC bash
sudo nixos-rebuild switch --flake .#navi
home-manager switch --flake .#sway
#+END_SRC

** Window Managers

*** XMonad (ThinkPad X220)
xmonad with =xmonad-contrib=. Layout is =Tall ||| Full= with screen-edge gaps and =avoidStruts= to respect the bar.

Keybinds:
| Key                    | Action                |
|------------------------+-----------------------|
| =Super + Enter=        | Launch Alacritty      |
| =Super + Shift + P=    | Rofi launcher         |
| =Super + Shift + C=    | Close window          |
| =Super + Q=            | Recompile and restart |

Keyboard layout: US with Caps Lock remapped to Ctrl (=ctrl:nocaps=).

*** Xmobar
Status bar with a dark theme. Displays workspace/layout info piped from xmonad, CPU usage, memory, battery states, and date/time. Spawned by xmonad via the =withSB= / =statusBarProp= API.

*** Sway (navi)
Wayland compositor with vim-style focus/move keybindings (h/j/k/l), inner/outer gaps, and the Autumn Frappe color palette. Autologin via greetd; screen locks with swaylock at 5 min idle, display off at 10 min (swayidle).

*** Waybar
Top bar for Sway: workspaces, window title, clock, CPU, memory, temperature, network, and audio. Themed with the Autumn Frappe palette.

*** Rofi
Application launcher on both profiles. On Sway, uses =rofi-wayland= (Wayland-native). Themed to match the Autumn Frappe palette with maroon accents.

*** Mako
Notification daemon for Sway. Autumn Frappe palette; high-urgency notifications persist until dismissed.

** Shared System Config

All hosts inherit =system/configuration.nix=, which provides:

- *Networking*: NetworkManager with =systemd-resolved= for DNS
- *VPN*: Mullvad (full GUI+CLI+daemon) with DNS kill switch; Tailscale mesh VPN with LAN bypass so Tailscale survives Mullvad's route changes
- *Firewall*: port 22 (SSH) and 41641 (Tailscale UDP); =tailscale0= fully trusted
- *Input method*: fcitx5 with =fcitx5-rime= and =fcitx5-chinese-addons= for Chinese
- *Shell*: fish (system-wide, default for =corinthian=)
- *GPG*: =gnupg.agent= with SSH support
- *Printing*: CUPS with Epson ESC/P-R driver
- *Fonts*: JetBrainsMono + Iosevka Nerd Fonts, Noto CJK, Source Han, custom icon fonts
- *Nix daemon*: weekly GC (7-day retention), auto-optimise store, flakes + ca-derivations, Garnix binary cache
- *Activation diff*: =nvd= diff report on every =nixos-rebuild switch=
- *Secrets*: SOPS-nix with age keys derived from each machine's SSH host key

** Virtualisation

- **KVM/QEMU** via =virt-manager= and =libvirtd=, with SPICE USB redirection; =quickemu= for lightweight VMs
- **Docker**: enabled system-wide, =corinthian= in docker group; log rotation (10 MB / 3 files)
- **nixos-rebuild build-vm**: =vmVariant= configured (4 GiB RAM, 4 cores)
