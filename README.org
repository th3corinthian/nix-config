This repository holds my personal NixOS configuration, managed with flakes for fully reproducible, declarative system management across two machines: a ThinkPad X220 and an AMD desktop. Host-specific files handle hardware and boot configuration, while a shared base covers services, user preferences, and home-manager integration — ensuring a consistent environment across different hardware.

* Hosts

** ThinkPad X220
Classic Intel laptop. Legacy GRUB on =/dev/sda=, latest kernel. Runs xmonad as the window manager with xmobar as the status bar.

To apply:
#+BEGIN_SRC bash
sudo nixos-rebuild switch --flake .#thinkpad-x220
home-manager switch --flake .#xmonad
#+END_SRC

** Tongfang AMD Desktop
AMD desktop in a Tongfang chassis. UEFI with systemd-boot, latest kernel. Runs XFCE.

Before first deploy, generate hardware config on the target machine:
#+BEGIN_SRC bash
nixos-generate-config --show-hardware-config > system/hosts/tongfang-amd/hardware-configuration.nix
#+END_SRC

Then apply:
#+BEGIN_SRC bash
sudo nixos-rebuild switch --flake .#tongfang-amd
home-manager switch --flake .#tongfang-xfce
#+END_SRC

* Desktop Environment

** XMonad (ThinkPad X220)
xmonad with =xmonad-contrib=. Layout is =Tall ||| Full= with screen-edge gaps and =avoidStruts= to respect the bar.

Keybinds:
| Key                    | Action               |
|------------------------+----------------------|
| =Super + Enter=        | Launch Alacritty     |
| =Super + Shift + P=    | Rofi launcher        |
| =Super + Shift + C=    | Close window         |
| =Super + Q=            | Recompile and restart |

Keyboard layout: US with Caps Lock remapped to Ctrl (=ctrl:nocaps=).

** Xmobar
Status bar with a matte dark blueish theme (=bg: #1c2028=). Displays:
- Workspace and layout info piped from xmonad via =statusBarProp=
- CPU usage, memory usage, battery (with charge/discharge/full states)
- Date and time

Spawned and managed by xmonad via the =withSB= / =statusBarProp= API.

** Rofi
Application launcher themed to match xmobar. Blue accent (=#7aa2d4=) on a dark slate background (=#1c2028=). Plugins: =rofi-calc=, =rofi-emoji=.

** Picom
Compositor running as a user systemd service.

* Shared System Config
All hosts inherit =system/configuration.nix=, which provides:

- *Networking*: NetworkManager, Mullvad VPN
- *Packages*: vim, git, alacritty, firefox, vlc, qbittorrent, ranger, android-tools, nasm, scrcpy
- *GPG*: =gnupg.agent= with SSH support
- *Docker*: enabled with weekly auto-prune and a custom bridge IP
- *Printing*: CUPS with Epson ESC/P-R driver
- *Shell*: fish (system-wide)
- *Nix daemon*: weekly GC (7-day retention), auto-optimise store, flakes + ca-derivations enabled
- *Activation diff*: =nvd= diff report on every =nixos-rebuild switch=

* Home Base
All home profiles share =home/shared/=, which includes:

- *Editors*: emacs
- *Dev tools*: cargo, rustc, gcc, gdb, gef, nodejs, nasm, cmake, libllvm, android-studio, claude-code
- *Security/network*: tor-browser, maltego, wireshark-adjacent tooling
- *Misc*: tmux (nord theme), syncthing, kicad, telegram-desktop, kitty, bottom, eza, fd, ripgrep, fzf, zoxide, bat, direnv, jq, fastfetch
